Overview
OpenHLTest is a collaborative effort by test vendors to develop agnostic programmatic interfaces and tools for managing test equipment.

For every Yang list, container, rpc and action keyword contained in the OpenHLTest models the model reference documents the following:
  • Yang model details
  • Python sample code
  • Python class
  • Restconf requests
To get the latest OpenHLTest python client you can install it using pip install -U openhltest or download it directly from pypi
